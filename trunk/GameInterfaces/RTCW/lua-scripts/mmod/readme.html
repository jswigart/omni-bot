About

Made to give RTCW easy to use admin commands.
Questions or help: goto www.thelunaticfam.com and post on the forum with your questions.
NOTE: mmod will only work in the OmniBot .82 mod for RTCW.

Installation

Extract the zip to your omnibot directory and add "mmod.lua" to your lua_modules cvar.
Setting your level for the first time, use /rcon !setlevel [name] [level]
Example: /rcon !setlevel mario 5

Configurable Variables

Open the /mmod/config.cfg file in a text editor (notepad)

slapdamage = 15 slap damage for !slap command
showkillerhp = 1 show your killer's hp 1 = on, 0 = off
kicktime = 2 minutes that player can rejoin after kick, player will get a temporary ban message on reconnect

Setting mmod Levels

To set levels open the /mmod/levels.cfg file in a text editor (notepad)

level = [level number]
name = [level's name]
flags = [flags defining permissions of this level] 		level = 0
name = Noob
flags = aCh

level = 1
name = Clan Friend
flags = aChitB

level = 2
name = Trusted Friend
flags = aChitBqmnAPRpf

level = 3
name = Semi Moderator
flags = aChitBqmnAPRgkNswToplSfxr

level = 4
name = Full Moderator
flags = aChitBqmnAPRgkNswTobpQlLSfxr

level = 5
name = King Admin
flags = *

Commands and Flags for mmod

COMMAND FLAG USAGE DESCRIPTION
!admindel Q !admindel [ADMIN NUMBER] removes the admin that !listadmins lists as ADMIN NUMBER.
!admintest a !admintest or !admintest [NAME|SLOT] prints out player's current mmod admin level in global chat. When the NAME or SLOT are added and the user has the right mmod permissions, the level of another player will be shown.
!announce f !announce [MSG] announces message in centerprint.
!ban b !ban [NAME|SLOT]
[TIME_WITH_MODIFIER] [REASON] 		Kicks the player in slot number SLOT or with name containing NAME off the server and will not let them connect again for time with modifier (s, m, h, d, or w). For example:

90s = 90 seconds
15m = 15 minutes
3h = 3 hours
5d = 5 days
1w = 1 week

If no time is specified then it is a permanent ban, example !ban [NAME|SLOT]
!botplayers o !botplayers [NUMBER OF BOTS] changes value of maxbots.
!date C !date prints out local date of the server in global chat.
!fling l !fling [NAME|SLOT] send player flying in slot number SLOT or with name containing NAME.
!flinga L !flinga send everyone flying
!gib g !gib [NAME|SLOT] [REASON] kills and gibs the player in slot number SLOT or with name containing NAME. If REASON is specified it is center-printed to the victim.
!help h !help or !help [COMMAND] lists all available mmod commands.
!kick k !kick [NAME|SLOT] [REASON] kicks the player in slot number SLOT or with name containing NAME off off the server. REASON is given to the client as the drop message. The client will be banned for 2 minutes.
!launch l !launch [NAME|SLOT] send player flying in slot number SLOT or with name containing NAME.
!launcha L !launcha send everyone flying
!listadmins q !listadmins or !listadmins [SKIP] lists up to 15 admins. columns are
admin number
admin level
admin level name
admin name
If SKIP is provided, admin number's less than that number are skipped before printing starts.
!listplayers i !listplayers prints out a list of information about the connected players. the columns are:
slot number
team R=axis B=allies S=spectator C=connecting
admin level (and level name)
last 8 digits of guid
M if muted
name
!lol x !lol [NAME|SLOT] make player drop grenades in slot number SLOT or with name containing NAME.
!map M !map [MAP NAME] change map to [MAP NAME]. i.e. !map mp_beach
!maprecords t !maprecords display a list of records for current map.
!mute m !mute [NAME|SLOT] [REASON] mute the player in slot SLOT or with name containing NAME. If REASON is specified it is center-printed to the victim.
!nextmap n !nextmap moves to the next map in rotation.
!putteam p !putteam [NAME|SLOT] [r|b|s] puts the player in slot number SLOT or with name containing NAME on the indicated team.
r - AXIS
b - ALLIES
s - SPECTATORS
!rename N !rename [NAME|SLOT] [NEW NAME] sets the player in slot number SLOT or with name containing NAME to NEWNAME.
!reset r !reset reset the match
!restart r !restart restart the map
!serverrecords t !serverrecords display a list of records for all maps on server.
!setlevel s !setlevel [NAME|SLOT] [LEVEL] sets the connected player in slot number SLOT or with name containing NAME to the level indicated by LEVEL. A player cannot !setlevel another player to a lever higher than him/herself.
!showbans B !showbans or !showbans [SKIP] lists up to 15 bans currently in effect. columns are
ban number
banned player's name
date of the ban
name of the banning admin
when the ban will expire
reason for the ban
If SKIP is provided, ban numbers less than that number are skipped before printing starts.
!shuffle S !shuffle shuffles teams
!slap A !slap [NAME|SLOT] [REASON] does damage (default 15) to player in slot SLOT or with name containing NAME. If REASON is specified it is center-printed to the victim.
!spec999 P !spec999 moves all players with ping of 999 to the spectators team.
!stats t !stats or !stats [NAME|SLOT] prints out your stats for current map. When the NAME or SLOT are added and the user has the right mmod permissions, the stats of another player will be shown.
!swap w !swap moves all players to the opposite team.
!time C !time prints out local time of the server in global chat.
!timelimit T !timelimit [MINUTES] changes the time limit of current map to MINUTES.
!throw l !throw [NAME|SLOT] send player flying in slot number SLOT or with name containing NAME.
!throwa L !throwa send everyone flying
!unban b !unban [BAN NUMBER] removes the ban that !showbans lists as BAN_NUMBER.
!unmute m !unmute [NAME|SLOT] unmute the player in slot SLOT or with name containing NAME.
!warn R !warn [NAME|SLOT] [WARNING] center-prints WARNING to the player in slot number SLOT or with name containing NAME.

*
admin has use of all commands.
